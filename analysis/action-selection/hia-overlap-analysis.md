# High-Impact Actions Overlap Analysis

**City:** Porto Alegre (BR POA)
**Date:** 2026-03-19
**Scope:** Cross-reference of three action sources to identify convergence zones for project preparation prioritization.

## Sources

| # | Source | Origin | Actions |
|---|--------|--------|---------|
| 1 | **C40 High Impact Actions (HIA)** | `high-impact-actions.csv` | 41 actions across 9 sectors |
| 2 | **HIAP API Assessment** | `HIA list (1).docx` | Same 41 C40 HIA actions (city-level assessment framework) |
| 3 | **Brazil Local Plans (PLAC + ProClima)** | `porto-alegre-energy-actions-plac-2024.yaml`, context docs | 8 energy actions (POA-E-06/07/08, RS-E-01/02/03/04, POA-RES-E-01) |

**Note:** The HIAP API list (source 2) contains the same 41 actions as the C40 HIA list (source 1). The HIAP is C40's implementation tracking framework — it uses the HIA catalog as its reference set. Therefore, the primary overlap analysis is between the **C40/HIAP framework** (41 actions) and the **Brazil/PLAC local actions** (8 energy-sector actions).

## Key Findings

### 1. Energy & Buildings is the strongest convergence zone

**11 of 41** C40/HIAP actions (IDs 10-16, 29, 30, 34, 39) have direct or indirect alignment with PLAC energy actions. This sector has:
- The most PLAC actions (POA-E-06, POA-E-07, POA-E-08)
- The best cost-effectiveness data (R$4-296/tCO2e range)
- Active implementation (PPP for 60 facilities, 10 biodigesters, PLS decree)

### 2. PLAC actions are narrower but deeper than C40 HIAs

The C40 list is broad (41 actions across 9 sectors). The PLAC energy actions focus on 3 city-level actions + 4 state-level enablers, but with detailed sub-actions, targets, and cost-effectiveness analysis. The PLAC provides the **implementation specificity** that C40 HIAs lack.

### 3. The HIAP API and C40 HIA are the same reference set

The HIAP docx contains the identical 41 actions as the C40 HIA CSV. HIAP is C40's assessment and tracking framework that uses the HIA catalog. For POA, the value of HIAP is in tracking **reporting progress** against these actions, not adding new ones.

### 4. Major gaps: Transport, Waste, Food, and Urban Planning

C40 HIAs 2-4, 7-9, 20-22, 24-27 have no energy-sector counterpart in PLAC. These are covered by other PLAC strategic axes (POA Mobilidade, POA Verde e Azul) but are outside the energy project preparation scope.

### 5. Highest-priority overlaps for project preparation

Ranked by convergence strength × cost-effectiveness × implementation readiness:

| Rank | Action | Why |
|------|--------|-----|
| 1 | **RS-E-03 → Solar policy enabling** | R$4/tCO2e, 225x leverage, already active. Maps to C40 HIA 12. |
| 2 | **POA-E-06 → Sustainable buildings** | R$9/tCO2e, 26,702 tCO2e/yr. Maps to C40 HIAs 10, 11, 14, 16. |
| 3 | **RS-E-02 → Biogas program** | R$43/tCO2e, dual waste+energy benefit. Maps to C40 HIA 39. |
| 4 | **POA-E-07 → Public buildings solar+biodigesters** | 100% municipal control, 37 MWp. Maps to C40 HIA 15. |
| 5 | **Electric bus fleet** | Already committed (100 buses). Maps to C40 HIAs 1, 5, 6. |
| 6 | **Grid resilience** | Post-disaster urgency, R$6B drainage. Maps to C40 HIAs 30, 31. |

## Recommendations for Project Preparation

1. **Focus energy project preparation on the 6 triple-convergence themes** — these have C40/HIAP validation, local plan commitment, and defined action archetypes.

2. **Use the C40/HIAP framework for international reporting** but rely on PLAC actions for implementation design — PLAC provides sub-actions, cost-effectiveness, targets, and institutional assignments that C40 HIAs do not.

3. **Address the building energy code gap** (C40 HIAs 10-11) — POA-E-06 proposes code updates but stops short of a mandatory net-zero pathway. This is an opportunity to strengthen alignment.

4. **The transport sector overlap is narrow** — only e-buses are covered. C40 HIAs 2-4, 7 (vehicle restrictions, road reallocation, speed limits) are not in PLAC energy scope. Cross-sector coordination needed.

5. **RS state policies (RS-E-03, RS-E-02) are force multipliers** — they provide the regulatory and market framework that makes municipal actions viable at scale. Project preparation should sequence state enablers before municipal capital investments.
